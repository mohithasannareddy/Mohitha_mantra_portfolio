# MANTRA 2026 Educational Website

## Project Overview
This project is a simple educational website developed using HTML, CSS, and JavaScript. It is designed as part of the MANTRA 2026 Summer School frontend assignment.

## Website Topic
Educational Website

## Technologies
- HTML5
- CSS3
- JavaScript

## Website Pages
- Home
- About
- Courses
- Gallery
- Contact

## Main Features
- Easy-to-use navigation
- Responsive layout for different screen sizes
- Course information page
- Image gallery
- Contact page
- Basic JavaScript functionality
- SEO-friendly page titles and meta tags

## Purpose
The aim of this project is to practice frontend web development by creating a responsive and user-friendly educational website.

## Running the Project
Download or clone the project files and open **index.html** in any web browser.

## Deployment
The project can be uploaded to GitHub and deployed using Netlify for public access.
